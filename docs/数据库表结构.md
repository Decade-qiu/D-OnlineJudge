## 用户

| 字段名         | 数据类型        | 描述             |
| -------------- | --------------- | ---------------- |
| `id`           | `bigint`        | NOT NULL, 主键   |
| `user_name`    | `varchar(12)`   | 用户昵称         |
| `email`        | `varchar(50)`   | 用户邮箱         |
| `password`     | `varchar(50)`   | 用户密码         |
| `score`        | `int`           | 用户刷题得分     |
| `rank`         | `int`           | 用户排名         |
| `school`       | `varchar(50)`   | 用户学校         |
| `gender`       | `varchar(10)`   | 用户性别         |
| `easy_solve`   | `int`           | 解决简单问题数量 |
| `middle_solve` | `int`           | 解决中等问题数量 |
| `hard_solve`   | `int`           | 解决困难问题数量 |
| `role`         | `varchar(50)`   | 用户角色         |
| `url`          | `varchar(1000)` | Github主页链接   |
| `sign`         | `longtext`      | 用户签名         |
| `fans`         | `bigint`        | 粉丝数量         |
| `subscribe`    | `bigint`        | 订阅数量         |
| `ban`          | `tinyint(1)`    | 用户是否被封禁   |
